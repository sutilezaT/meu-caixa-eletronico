# menu-caixa-eletronico

Exercício para prática de switch. 

O Problema: Você está criando a interface de texto para um caixa eletrônico. O programa deve apresentar um menu e reagir à escolha do usuário.

Requisitos de Negócio:
O nome do programa (e da classe principal) deve ser MenuCaixaEletronico.
O programa deve, primeiro, exibir as seguintes opções para o usuário (exatamente como abaixo):
  1. Ver Saldo
  2. Depositar Valor
  3. Sacar Valor
  4. Sair
Em seguida, o programa deve solicitar que o usuário digite o número da opção desejada.

O programa deve ler essa escolha.
O programa deve então exibir uma mensagem de confirmação específica para cada escolha:

Se a escolha for 1, a mensagem deve ser: "Seu saldo atual é de R$ 0,00."
Se a escolha for 2, a mensagem deve ser: "Operação de depósito selecionada."
Se a escolha for 3, a mensagem deve ser: "Operação de saque selecionada."
Se a escolha for 4, a mensagem deve ser: "Obrigado por usar nosso banco. Saindo do sistema..."

Tratamento de Erro: Se o usuário digitar qualquer número que não seja 1, 2, 3 ou 4, o programa deve exibir a mensagem: "Opção inválida. Por favor, tente novamente."

Esse código será refinado e implementado em outro projeto no futuro.
